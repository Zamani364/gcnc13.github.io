## How to Host on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to repository Settings > Pages.
4. Select Deploy from a branch.
5. Select main branch and /root folder.
6. Save and open the GitHub Pages link.

## Suggestions Storage
GitHub Pages cannot run PHP or MySQL. The suggestion form saves suggestions in the visitor's browser using localStorage by default.

For online storage, use google-apps-script-example.js to connect the form to Google Sheets.
